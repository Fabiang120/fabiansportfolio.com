# Fabian's Portfolio

A modern, interactive portfolio website built with Next.js, featuring 3D elements and smooth animations.

## Prerequisites

Before running this project, make sure you have **pnpm** installed on your machine.

### Installing pnpm

If you don't have pnpm installed, you can install it using one of these methods:

```bash
# Using npm
npm install -g pnpm

# Using Homebrew (macOS)
brew install pnpm

# Using Scoop (Windows)
scoop install pnpm
```

For more installation options, visit the [pnpm installation guide](https://pnpm.io/installation).

## Getting Started

### 1. Navigate to the portfolio directory

```bash
cd portfolio
```

### 2. Install dependencies

```bash
pnpm install
```

### 3. Run the development server

```bash
pnpm run dev
```

### 4. Open in browser

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org)
- **3D Graphics**: React Three Fiber
- **Animations**: Framer Motion
- **Styling**: CSS Modules
